# MovieSearch - Flask Web App

Web application for searching movies in Sakila database with MongoDB analytics.

## Features
- Search by title keyword (with pagination)
- Filter by genre & year range  
- MongoDB statistics (top 10 popular, top 5 recent queries)

## Technologies
- Flask 3.0.0, PyMySQL, PyMongo
- MySQL (Sakila), MongoDB (logging)

## Quick Start
```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
